## 🚀 What yilan does

yilan is a Chromium extension that helps you read web pages faster. It can turn long articles into short summaries, split long content into parts, keep your history, and save share cards.

It is built for people who want a simple way to use AI on web pages without leaving the browser. It works with OpenAI and Anthropic compatible APIs, and it also supports local use with your own key.

## 🪟 Install on Windows

1. Open the releases page.
2. Download the latest Windows package or extension file.
3. If the file is a ZIP folder, right-click it and choose **Extract All**.
4. Open Chrome, Edge, or Brave.
5. Go to the extensions page:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
6. Turn on **Developer mode** in the top right.
7. Click **Load unpacked**.
8. Pick the yilan folder you extracted.
9. Pin the extension if you want quick access from the browser bar.

If the release gives you a packaged extension file, follow the same browser steps and load the unpacked folder that came with it.

## 🖥️ How to use yilan

1. Open a web page you want to read.
2. Click the yilan extension icon.
3. Choose the summary mode you want.
4. Let yilan process the page.
5. Read the summary or open the reader view.
6. Save the result if you want to keep it.
7. Copy or export the text in Markdown if needed.

For long articles, yilan can break the content into smaller parts. This helps when a page is too large for one pass.

## 🧩 Troubleshooting

### The extension does not show up
- Open the extensions page again
- Check that Developer mode is on
- Make sure you loaded the correct folder
- Refresh the browser after loading it

### The summary does not appear
- Check your API key
- Confirm the model name is correct
- Make sure your endpoint URL is valid
- Try again on a simpler page

### The page looks broken in reader mode
- Refresh the page
- Open the extension again
- Try a different article
- Use a page with normal text content

### The AI request fails
- Check your internet connection
- Check your API balance or quota
- Confirm that your provider accepts the model you chose
- Review the base URL if you use a custom service
